Escriba una librería javascript que permita autocompletar la entrada de un elemento input de texto que tengan clase “autocomplete”. input type=”text” class=”autocomplete”

La librería debe tener en un array su propia base de datos de palabras con las que puede autocompletar.

A medida que el usuario va agregando una letra a la caja de texto la librería debe ir proponiendo las palabras cuyo comienzo coincidan con lo escrito por el usuario
